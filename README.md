## APIs
### API 1
**Get Request to https://api.devsite.com/api**

  `min` Minimum font-size in pixels  
  `max` Maximum font-size in pixels  
  `mid` Mid-range buffer

  ```javascript

  (function($) {

    $.fn.dogsay = function(content) {
      
         $(this).html(content);
      
      };

  })(jQuery);

  ```

**API 1 Response Example**  
```javascript
{data[0]}
```
